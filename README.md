# Customer-Churn-Prediction
End-to-end MLOps pipeline that predicts if a customer is likely to leave a telecom company

Requirements (Core Libraries to install):

1.) scikit-learn
2.) xgboost
3.) mlflow
4.) fastapi
5.) uvicorn
6.) docker
7.) evidently
8.) pytest
